# 北平城春 - 火种

一个简单的剧本杀网页游戏，主题为1949年北平城的地下工作。

## 功能特点

- 🎭 三个可选角色：学生领袖、进步记者、爱国商人
- 🤖 集成DeepSeek AI，提供智能对话体验
- 🎯 选项系统：每次AI会提供2-3个选项，引导剧情发展
- 📊 积分系统：根据选择获得或失去积分
- 🏆 多结局系统：根据积分和回合数判定不同结局
  - 完美结局（80+分）：火种燎原
  - 良好结局（60+分）：星星之火
  - 普通结局（40+分）：风雨飘摇
  - 失败结局（<40分）：火种熄灭
- 🎨 复古风格的UI设计，符合历史背景
- 📱 响应式设计，支持移动端

## 游戏玩法

1. 直接在浏览器中打开 `index.html` 文件
2. 阅读背景故事，点击"开始游戏"
3. 选择一个角色（学生领袖/进步记者/爱国商人）
4. 输入你的名字开始游戏
5. AI会给出剧情描述和2-3个选项
6. 点击选项按钮做出选择
7. 根据选择获得积分，推进剧情
8. 游戏进行8回合后或达成结局条件时结束
9. 根据最终积分和表现获得不同结局

## 积分规则

- 优秀选择：+10到+15分
- 一般选择：+5到+10分
- 错误选择：-5到0分

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- DeepSeek API

## 注意事项

- 需要网络连接以调用DeepSeek API
- API密钥已内置在代码中，如需更换请修改 `script.js` 中的 `DEEPSEEK_API_KEY`
- 游戏最多进行8回合
- 积分和选择会影响最终结局
